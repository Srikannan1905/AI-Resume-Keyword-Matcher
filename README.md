# 🚀 AI Resume Keyword Matcher

An AI-powered tool to **analyze resumes against job descriptions** and highlight **missing keywords, strengths, and improvement areas**. Built with a modern **glassmorphism UI** for a premium user experience.

---

## 📋 Project Overview
The **AI Resume Keyword Matcher** helps job seekers optimize their resumes for **Applicant Tracking Systems (ATS)**. It performs a deep contextual analysis, identifying skill gaps and providing a compatibility score with actionable recommendations.

---

---

## 🧠 AI & Matching Model
The project utilizes a custom-built **Natural Language Processing (NLP) Lite** engine that runs entirely client-side for maximum privacy and speed.

### Key Logic Phases:
1.  **Text Normalization**: Regex-based cleaning and stop-word removal.
2.  **Tokenization**: Segmenting text into searchable units.
3.  **Heuristic Categorization**: Mapping keywords to domains like **Cloud/DevOps**, **Programming Languages**, and **Soft Skills**.
4.  **Compatibility Matrix**: 
    *   **Exact Match**: Direct keyword mapping.
    *   **Partial Match**: Contextual relevance checks.
    *   **Priority Scoring**: Weighted analysis favoring technical core skills.

## ✨ Features
- 📂 **Multi-format Support**: Upload resumes in **PDF, DOCX, or TXT**.
- 📋 **JD Analysis**: Paste any job description to extract core requirements.
- 📊 **Visual Analytics**: Interactive bar charts showing skill distribution.
- 📈 **ATS Score**: Animated progress ring showing overall match percentage.
- 💡 **AI Recommendations**: Specific tips on how to improve your score.
- 📥 **Export Reports**: Save your analysis as **JSON** or a formatted **Text Report**.
- 🤖 **Interactive Assistant**: Built-in chatbot to answer tool-related questions.
